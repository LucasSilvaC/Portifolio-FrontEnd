Funções em Javascript

Em programação, uma função é um bloco de código que executa uma tarefa específica. Pode ser reutilizada sempre que necessário, evitando a repetição de código. Existem dois tipos de funções as arrow functions e as funções normais, abaixo estão alguns exercicios que utilizam algumas delas:

Function

Uma função normal em JavaScript é declarada usando a palavra-chave function e pode ter seu próprio contexto de parâmetros. Ela é mais flexível, mas geralmente mais verbosa do que as arrow functions.

Terminal - Simulação
function soma(a, b) {
return a + b;
}

Arrow Function

Uma arrow function é uma forma mais concisa de escrever funções em JavaScript, usando a sintaxe () =>. Ela não possui seu próprio parâmetro e é vazia, herdando-o do contexto onde foi definida.

Terminal - Simulação
const soma = (a, b) => a + b;
