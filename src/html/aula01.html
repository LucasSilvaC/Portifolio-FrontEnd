Usabilidade

Usabilidade é a facilidade com que um usuário consegue interagir com um produto ou sistema. É um conceito fundamental no design de produtos e serviços, especialmente no contexto digital.

5 Conceitos de usabilidade

Ao pensar em usabilidade você deve se fazer 5 perguntas para avaliar seu sistema, seguindo alguns princípios, que são:

  1. Aprendizado - Quão fácil é para um usuário aprender a usar o sistema pela primeira vez?
  2. Eficiência – Depois de aprender, o usuário consegue realizar tarefas rapidamente?
  3. Memorabilidade – Se um usuário ficar um tempo sem usar o sistema, ele conseguirá lembrar como utilizá-lo sem precisar reaprender tudo?
  4. Erros – Quantos erros os usuários cometem, quão graves são esses erros e quão fácil é corrigi-los?
  5. Satisfação – O uso do sistema é agradável?
