O que é um Array

Um array é uma estrutura de dados que armazena vários valores em uma única variável. Também é conhecido como arranjo ou vetor.

Arrays Callback

Um callback é uma função que testa cada elemento de um array, e um array é uma estrutura que armazena dados passada como argumento para outra função. Alguns exemplos de arrays nessa ordem é: Callback, Find e Filter.
